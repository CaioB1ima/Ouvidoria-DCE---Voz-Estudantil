Sistema de Ouvidoria desenvolvido para o DCE, permitindo que estudantes registrem e acompanhem manifestações com segurança e praticidade. O projeto inclui login institucional com código de verificação, painel administrativo exclusivo, FAQ integrado, exibição de parcerias e interface moderna com carrossel de imagens.

Construído com React + Vite no frontend e Node.js + Express + tRPC no backend, utilizando MySQL com Drizzle ORM para gerenciamento de dados, além de autenticação com JWT e BCrypt.

COMO EXECUTAR:

pnpm install
pnpm db:push
pnpm dev
